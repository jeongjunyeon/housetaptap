 # 프로젝트 소개
 
 🌱 House Tap Tap 프로젝트는 인테리어를 원하는 고객에게 회원가입, 로그인, 문의 서비스를 통해서 
 빠른 인테리어를 제공하는 프로젝트입니다!
 
 + 제가 이 프로젝트를 만든 이유는 저는 과거에 인테리어 시공을 했습니다. 물론 일도 힘들지만 가장 어려웠던건 영업이었습니다. 
 + 그래서 고객과 시공사를 연결해줄 수 있는 웹 프로젝트를 기획하였습니다.
